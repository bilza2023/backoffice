
Here are the rules for question.status in this backoffice app

1. unlocked : is visible to any teacher.
2. Fill: once has some data and is saved once it will have status = fill. The teacher can just access the fill (just those which are filled by him) throught filledByMe page and can edit it.
3. Locked: Once a question is considered complete from the point of view of input it is marked Locked by admin and should not be edited any more.
4. Final: in this stage we add narration and timings
